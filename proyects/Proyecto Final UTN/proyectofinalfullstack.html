Proyecto final

Objetivo Diseñar un sitio web que contenga 5 secciones, de las cuales una será la página inicial de nuestro sitio, tres secciones relacionadas al tema seleccionado y una cuarta sección de contacto. Requisitos:Maquetación sencilla del sitio con CSS, se puede utilizar algún framework del estilo de Bootstrap si lo desean. Incluir algún componente oplugin de JQueryen alguna de las secciones. Realizar mediante NodeJS el manejo de las rutas del sitio. Pueden usar Jade, Handlebars, o cualquier otro sistema de plantillas o Angular. Dosde estas secciones debentener interacción con la base de datos para lo que deberán incluir los paquetes necesarios asociados a la base de datos con la que están trabajando, ya sea Mysql o MongoDB. Una sección tiene que acceder a alguna APIo paquete propioque devuelva un JSON. Almenos una de las secciones tiene que tener el uso de Ajax. Debe tener un formulario de contacto validado y tiene que hacerel envio de maila través de NodeMailer o de la API de Gmail. Formato de entrega El T.P. deberá ser entregado en el siguiente formato:Una carpeta con apellido y nombre del alumno que contenga:Todas las carpetas y archivos necesarios para el correcto funcionamiento del sitioincluyendo lassubcarpetas adicionales que se hayan utilizado comopor ejemplo css, imágenes, etc y el script correspondiente a la Base de datos.
