Nombre del proyecto: Sensorit

Introduccion

El proyecto consiste en una webapp que muestra la respuesta de diferente tipo de sensores: temperatura, humedad, movimiento, luz, etc. El usuario puede dar de alta un sensor y luego ver las mediciones donde se encuentre. Para esto utilice la plataforma de open hardware Arduino y sensores conectados a esta plataforma. La medicion de los sensores es en tiempo real por lo tanto utilice websockets en lugar de Ajax porque necesitaba tener una comunicacion continua entre la plataforma y Arduino.

Programacion del Arduino

La programacion del arduino es bastante sencilla y la realice con el IDE de Arduino

Objetivo

Diseñar un sitio web que contenga 5 secciones, de las cuales una será la página inicial de nuestro sitio, tres secciones relacionadas al tema seleccionado y una cuarta sección de

Requisitos:

Maquetación sencilla del sitio con CSS, se puede utilizar algún framework del estilo de Bootstrap si lo desean. Incluir algún componente oplugin de JQueryen alguna de las secciones. Realizar mediante NodeJS el manejo de las rutas del sitio. Pueden usar Jade, Handlebars, o cualquier otro sistema de plantillas o Angular. Dosde estas secciones debentener interacción con la base de datos para lo que deberán incluir los paquetes necesarios asociados a la base de datos con la que están trabajando, ya sea Mysql o MongoDB. Una sección tiene que acceder a alguna APIo paquete propioque devuelva un JSON. Almenos una de las secciones tiene que tener el uso de Ajax. Debe tener un formulario de contacto validado y tiene que hacerel envio de maila través de NodeMailer o de la API de Gmail.
